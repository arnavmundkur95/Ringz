Download Eclipse.
Download project zip file.
Go to File, Open Projects from File system
Navigate to the zip file.


Once the folder is loaded in the package explorer:
Double click on src.
Double click on controllers.
Double click on Server so that the class opens.
Double click on Client so that the class opens.

Run the server class.
Run the client class as many times as you want other players.

Answer the question in the console to play the game.
